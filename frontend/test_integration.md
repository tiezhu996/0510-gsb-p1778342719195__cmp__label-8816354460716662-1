# 前后端联调测试指南

## 测试步骤

### 1. 启动服务

#### 后端服务
```bash
cd backend
python run.py
# 或
.\venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

#### 前端服务
```bash
cd frontend
npm run dev
```

### 2. 测试流程

#### 2.1 访问前端
- 打开浏览器访问：http://localhost:5173
- 应该自动重定向到登录页面：http://localhost:5173/login

#### 2.2 测试注册功能
1. 点击"立即注册"或访问 http://localhost:5173/register
2. 填写注册表单：
   - 用户名：testuser（3-50个字符）
   - 邮箱：test@example.com
   - 密码：test123456（至少6个字符）
   - 确认密码：test123456
3. 点击"注册"按钮
4. **预期结果**：
   - 注册成功
   - 自动登录
   - 跳转到聊天页（/chat）
   - 能看到聊天室列表与在线用户区域

#### 2.3 测试登录功能
1. 点击"登出"按钮
2. 应该跳转到登录页面
3. 填写登录表单：
   - 用户名或邮箱：testuser（或test@example.com）
   - 密码：test123456
4. 点击"登录"按钮
5. **预期结果**：
   - 登录成功
   - 跳转到聊天页（/chat）
   - 能看到聊天室列表与在线用户区域

#### 2.4 测试错误处理
1. **错误密码测试**：
   - 输入错误的密码
   - 应该显示错误提示："用户名或密码错误"

2. **未注册用户测试**：
   - 输入不存在的用户名
   - 应该显示错误提示："用户名或密码错误"

3. **注册重复用户测试**：
   - 尝试注册已存在的用户名或邮箱
   - 应该显示错误提示："用户名已存在"或"邮箱已被注册"

#### 2.5 测试Token持久化
1. 登录成功后，刷新页面（F5）
2. **预期结果**：
   - 应该保持登录状态
   - 不需要重新登录

#### 2.6 测试路由保护
1. 在未登录状态下，直接访问 http://localhost:5173/chat
2. **预期结果**：
   - 应该自动重定向到登录页面

### 3. 检查点

- [ ] 登出功能正常

- [ ] 加载状态正确显示

### 4. 浏览器开发者工具检查

打开浏览器开发者工具（F12），检查：

1. **Network标签**：
   - 注册请求：POST /api/auth/register
   - 登录请求：POST /api/auth/login
   - 获取用户信息：GET /api/auth/me
   - 请求头中应该包含：`Authorization: Bearer <token>`

2. **Application标签**：
   - Local Storage中应该存储：
     - `auth_token`: JWT token
     - `user_info`: 用户信息JSON

3. **Console标签**：
   - 不应该有错误信息

### 5. 常见问题

#### 问题1：CORS错误
**解决方案**：检查后端CORS配置，确保允许前端域名

#### 问题2：401未授权错误
**解决方案**：
- 检查Token是否正确存储
- 检查请求头是否正确添加Token
- 检查后端Token验证逻辑

#### 问题3：无法自动登录
**解决方案**：
- 检查localStorage中是否有Token
- 检查AuthContext的初始化逻辑

#### 问题4：路由跳转问题
**解决方案**：
- 检查router.tsx配置
- 检查ProtectedRoute组件逻辑
